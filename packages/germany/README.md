# Germany

Contains all the states (_Länder_) of Germany:
* Baden-Württemberg
* Bavaria
* Berlin
* Brandenburg
* Bremen
* Hamburg
* Hesse
* Lower Saxony
* Mecklenburg-Vorpommern
* North Rhine-Westphalia
* Rhineland-Palatinate
* Saarland
* Saxony
* Saxony-Anhalt
* Schleswig-Holstein
* Thuringia

## Installation

### npm

`npm install --save @svg-maps/germany`

### yarn

`yarn add @svg-maps/germany`
