# SY Syria

Contains all the governorates of Syria:

- Aleppo
- As-Suwayda
- Darra
- Damascus
- Deir ez-Zor
- Hama
- Hasaka
- Homs
- Idlib
- Latakia
- Quneitra
- Raqqa
- Rif Dimashq
- Tartus

## Installation

### npm

`npm install --save @svg-maps/syria`

### yarn

`yarn add @svg-maps/syria`
