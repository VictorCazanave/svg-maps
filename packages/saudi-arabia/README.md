# sa Saudi Arabia

Contains all the regions of Saudi Arabia:
* Ar Riyāḑ
* Makkah
* Al Madīnah
* Ash Sharqīyah
* Al Qaşīm
* Ḩā'il
* Tabūk
* Al Ḩudūd ash Shamālīyah
* Jīzān
* Najrān
* Al Bāḩah
* Al Jawf
* Asir

## Installation

### npm

`npm install --save @svg-maps/saudi-arabia`

### yarn

`yarn add @svg-maps/saudi-arabia`
