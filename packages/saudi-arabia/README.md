# SA Saudi Arabia

Contains all the regions of Saudi Arabia:
* Asir
* Al Bahah
* Al Hudud al Shamaliyah
* Al Jawf
* Al Madinah
* Al Qasim
* Al Ryiadh
* Al Sharqiyah
* Ha'il
* Jizan
* Makkah
* Najran
* Tabuk

## Installation

### npm

`npm install --save @svg-maps/saudi-arabia`

### yarn

`yarn add @svg-maps/saudi-arabia`
